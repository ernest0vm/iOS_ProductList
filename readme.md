## Instructions to run project

open a terminal and clone repository

```
git clone https://github.com/ernest0vm/iOS_ProductList.git
```

after type command to navigate into project folder

```
cd iOS_ProductList
```

then run pod command for install all dependencies

```
pod install
```

for open the project in xcode 11 find into the folder the project file:

**iostest.xcworkspace**
